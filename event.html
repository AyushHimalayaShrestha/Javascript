onclick -> when click on the html element.
        ondblclick -> double click on the element.
        onload -> when a page is loaded.
        onsubmit -> when a form is submitted.
        onchange -> when the input field is changed.
        onmouseover -> when a mouse moves over an element.
        onmouseout -> when a mouse moves out from an element.
        onkeyup -> when a key is released after press.
        onkeydown -> when a key is down.
